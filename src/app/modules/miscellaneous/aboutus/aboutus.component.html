About Us

The home page will automatically sign you in with email 'user@gmail.com' and password ‘user1234’ for you to test out and look at various features of the website. Developed dynamic e-commerce platform with Angular frontend and Spring Boot REST API backend. Used Bootstrap as the Utility CSS Framework and Angular Material as the Component Library. Deployed SPA on Vercel, RESTful API on Azure, utilizing Azure SQL Database. Implemented Spring Security with JWT for data integrity and user privacy. Customer Experience: • Implemented pagination, sorting and filtering for the products on the home page. • Seamlessly integrated search for product exploration such that it can work in sync with filtering and also independent of filtering. • Streamlined shopping experience with cart management, enabling users to add, edit or remove products. • Facilitated payments through a seamless checkout process. • Implemented Orders page for tracking previous purchases and a Wishlist for saving desired items. Admin facing side: • Facility to add/delete/update any product. • All Orders page that displays a clickable list of all the orders placed on the website and total earnings. • Order Details page that lists all of the products and their quantity in the order. • Enhanced decision-making with detailed Wishlist Data for analysing product popularity.
